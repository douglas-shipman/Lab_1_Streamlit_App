# Movie Review Sentiment Analyzer

This Streamlit application predicts whether a movie review has positive or negative sentiment
using a TF-IDF vectorizer and Multinomial Naive Bayes classifier.

## Run Locally

- Clone the repository.
- Open a terminal in the project folder.
- Create and activate a virtual environment:

  ```bash
  python3 -m venv .venv
  source .venv/bin/activate
